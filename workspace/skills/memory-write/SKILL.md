# Memory Write Skill

When the user shares important information that should be remembered across future conversations (such as: important life events, preferences, goals, relationships, dates), call this skill to write it to `MEMORY.md`.

## When to use:

- User shares significant life events (achievements, losses, changes, etc.)
- User states clear preferences they want you to remember
- User talks about important goals they're working toward
- User mentions important people in their life
- Any other information that would be useful to remember in future conversations

## When NOT to use:

- For temporary context of the current conversation
- For trivial details that don't matter long-term
- When the user asks to forget something (remove it from MEMORY.md instead)

## How to use:

When you identify something important to remember, add it as a bullet point to `MEMORY.md` with a clear, concise description.
